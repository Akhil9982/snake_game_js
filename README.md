Snake Game JS 🐍

A classic Snake Game built using Vanilla JavaScript, HTML, and CSS, bundled with Vite.

🚀 Live Demo

Play the Game￼

Link: https://akhil9982.github.io/snake_game_js/

⸻

✨ Features

* Snake movement using keyboard controls
* Food generation system
* Score tracking
* High score stored using localStorage
* Pause / Resume functionality
* Game Over modal
* Restart game option
* Self-collision detection
* Wall collision detection
* Responsive game board

⸻

🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Vite

⸻

🎮 Controls

Key	Action
⬆️ Arrow Up	Move Up
⬇️ Arrow Down	Move Down
⬅️ Arrow Left	Move Left
➡️ Arrow Right	Move Right

⸻

📦 Installation

Clone the repository:
```bash
git clone https://github.com/Akhil9982/snake_game_js.git
```
Navigate to the project folder:
```bash
cd snake_game_js
```
Install dependencies:
```bash
npm install
```
Start development server:
```bash
npm run dev
```
⸻

🏗️ Build For Production
```bash
npm run build
```
⸻

🚀 Deploy To GitHub Pages
```bash
npm run deploy
```
